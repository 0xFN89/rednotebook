Ubuntu

  • RedNotebook is available from the official Ubuntu repositories; click here to install it. Ubuntu has a policy of not updating the version of a shipped application, so the version available might be old.
  • To keep up to date with the latest version, add the RedNotebook repository. Please note that it takes some time before a new version shows up in the repo. We want to test new debs thoroughly before uploading them.
  • You can also download the latest deb package directly from the repository.

Fedora

  • Install via yum: yum install rednotebook

Debian

  • Debian Testing: apt-get install rednotebook

Windows

  • Get the latest Installer or try RedNotebookPortable.
  • (Close RedNotebook before updating, if you experience Error 14001, read here)
  • Please note that it may take some time before the version of the installer catches up with the last RedNotebook source release
  • The development of the Windows version and Windows specific support take lots of time, so please consider a small donation.

Mac OS X

  • There is a MacPort for RedNotebook (you may have to uninstall Fink and reinstall Macports).
  • You can also find instructions on how to build and run RedNotebook on OS X at softpedia and here.

Source

  • Download the latest source package, unpack it and in a terminal window:
$ cd rednotebook
$ sudo apt-get install python-yaml python-webkit python-chardet
$ python rednotebook/journal.py
