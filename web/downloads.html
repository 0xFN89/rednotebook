Select distribution

Ubuntu

  • You can find RedNotebook in the Ubuntu Software Center or install it directly.
  • Add the RedNotebook repository to always get the latest version automatically (detailed instructions, you can safely ignore the warnings): 
    sudo add-apt-repository "deb https://robin.powdarrmonkey.net/ubuntu maverick/"
sudo apt-get update
sudo apt-get install rednotebook powdarrmonkey-keyring
  • Alternatively download the latest deb package directly from the repository.
  • Please note that it might take some time before the latest release is packaged and added to the repository.

Fedora

  • yum install rednotebook

Debian

  • apt-get install rednotebook

Windows

  • Keep RedNotebook development going by making a small donation:

in US Dollars in Euros

  • Get the latest Installer or try a portable version from PortableApps or winPenPack.
  • You must close RedNotebook before updating. If you experience Error 14001, read here.
  • Please note that it may take some time before the version of the installer catches up with the last RedNotebook source release.

Mac OS X

  • There is a MacPort for RedNotebook (you may have to uninstall Fink and reinstall MacPorts).
  • Alternatively, you can find instructions on how to build and run RedNotebook on OS X at softpedia and here.

Source

  • Download the latest source package, unpack it and in a terminal window:
$ cd rednotebook
$ sudo apt-get install python-yaml python-webkit python-chardet
$ python rednotebook/journal.py
