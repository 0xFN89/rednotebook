Download RedNotebook

There are RedNotebook packages for many distributions. If there is no package for your distribution, just download the source package, unpack it and follow the instructions in the README file. It’s easy

If you have any trouble downloading or installing, please drop me a note either in the forum or per mail.

Jonathan Wiltshire (the RedNotebook Debian and Ubuntu packager) has created a nice map showing where RedNotebook users come from (The map shows where the ISP’s (Internet Service Providers) of the Ubuntu RedNotebook users are located).

Source: Download the latest source package.
Ubuntu: Install Now, Latest Deb Packages and Repository
Arch Linux: http://aur.archlinux.org/packages.php?ID=21984
Debian: sudo apt-get install rednotebook (http://packages.debian.org/sid/rednotebook)
Fedora: yum install rednotebook (Package at fedoraproject.org)
Slackware: slackbuilds.org, slacky.eu
Gentoo: http://packages.gentoo.org
OpenSUSE: Instructions
Windows: Get the latest Installer or try RedNotebookPortable. (Close RedNotebook before updating) (If you experience Error 14001, read here)
OpenBSD: Package at www.openbsd.org
Mac OS X: You can find instructions on how to build and run RedNotebook on OS X at softpedia and here (Please tell me if it works).

Older Versions:

You can also browse through older versions.

Launch

  • If you downloaded the archive, unzip it and read ‘README’
  • Ubuntu/Debian: You can find a menu entry in the main menu under ‘Office’
  • Other *nixes: Press “Alt-F2″, type “rednotebook” and hit enter
  • Windows: You find the program under “Start -> Programs -> RedNotebook”
